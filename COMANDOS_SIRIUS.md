# Comandos do Sirius - Assistente de Voz

## Como usar
- Pressione **Ctrl+Shift+Espaço** para ativar o assistente
- Pressione **L** para ativar o **Modo Live** (microfone sempre aberto)
- Pressione **Shift+Escape** para **Background Mode** (2 palmas para ativar)
- Pressione **Escape** para sair

## Resumo de Comandos Principais

### Abrir/Executar
- "Abre o [app]" - Abre qualquer aplicativo
- "Toca [música]" - Toca no YouTube
- "Pesquisa [termo]" - Pesquisa na web

### Controlar
- "Fecha [app]" - Fecha aplicativo
- "Volume [0-100]" - Ajusta volume
- "Foca no [app]" - Foca janela

### Informar
- "Que horas são?" - Hora atual
- "Como está o tempo?" - Clima
- "O que está aberto?" - Apps rodando

### Comunicar
- "Manda mensagem pro [nome]" - WhatsApp
- "Me lembra às [hora]" - Lembrete

### Memória
- "Lembra que [info]" - Salva na memória
- "O que você lembra?" - Lista memórias

## Tips

- **Sea direto**: "Abre Discord" funciona melhor que "Pode, por favor, abrir o Discord?"
- **Modo Live (L)**: Quando ativado, o assistente ouve automaticamente
- **Interrompa**: Pressione Ctrl+Shift+Espaço durante resposta para interromper
- **Two-clap**: Duas palmas ativam o modo background
